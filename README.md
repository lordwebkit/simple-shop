# Simple-shop

This shop is delevopmented on Vue 3 + Vuex + Vue-router

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Format for Vue 3 recomends style guide

```sh
npm run format
```
